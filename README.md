# Prisoner-Escape-Master
Prisoner Escape Master is a DAA based game project.The nodes are the prison and the edges are the paths connecting the prsions
In this the prisoner tries to escape from prison by finding the shortest path to exit using Dijkstra's Algorithm. The user acts as a a police and blocks  edges under some fixed rules and time limit. The game includes two timer, making the escape challenging  and strategic.
There is only start and exit point and there are two timers to maintain the fairness in the gameplay. The first is the main game timer which is of 5 mins and the second is a cooldown timer which is of 5 sceonds and while the cool down timer is not over till that time the user action of blocking the edges will be freezed.
If the main game timer is over and the prisoner is not able to reach the exit point then police(we)wins otherwise the prsioner wins if ut raeched the exit point by satisfying all the conditions.
